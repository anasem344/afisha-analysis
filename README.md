# 🎟️ Yandex Afisha Ticket Sales Analysis

## 📌 Описание проекта

Проект посвящён анализу данных сервиса продажи билетов на мероприятия.  
Цель — изучить поведение пользователей, выявить ключевые драйверы выручки и оценить влияние сезонности, сегментов и цен на продажи.

Анализ выполнен на данных за период **июнь–октябрь 2024 года**.

---

## 🛠️ Используемые инструменты

- SQL — первичный анализ и проверка данных  
- Python (Pandas, Matplotlib) — предобработка и исследовательский анализ  
- DataLens — построение интерактивного дашборда
